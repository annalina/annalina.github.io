Projects

  • Archives in the UK/ Republic of Ireland & AI (AURA): AURA is a networking projected funded by the Irish Research Council and the Arts and Humanities Research Council (AHRC) in the UK.
  • Kronos-it: a Dataset for the Italian Semantic Change Detection Task (Kronos-it): A dataset for the evaluation of semantic change point detection algorithms for the Italian language. The dataset is automatically built by using a web scraping strategy, described in (Basile et al., 2019). Available under “GNU General Public License v3.0”.
  • Diachronic Analysis of Entities by Exploiting Wikipedia Page revisions (DAE): A tool for performing the diachronic analysis of named entities built upon Wikipedia page revisions, described in (Basile et al., 2019). Available under “GNU General Public License v3.0”
  • Time-Aware Information Retrieval (TAIR): A system for Information Retrieval that enables the expression of temporal constraints, described in (Basile et al., 2017). Available under “GNU GENERAL PUBLIC LICENSE Version 3”.
  • Temporal Random Indexing (TRI): A system for the analysis of meaning variations over time, described in (Basile et al., 2015). Available under “GNU GENERAL PUBLIC LICENSE Version 3.
  • Lesk Word Sense Disambiguation Distributional Semantic Model (Lesk-WSD-DSM): A Lesk-based disambiguation Algorithm, described in (Basile et al., 2014). Available under “GNU GENERAL PUBLIC LICENSE Version 3”.
  • Quantum Information Retrieval framework (QIR) I contributed to the quantum query algebra for query representation, described in (Caputo et al., 2011).
  • SEmantic N-levels Search Engine (SENSE) Described in (Basile et al., 2008).
